# 🔐 SPDF Reader (Secure Protected Document Format)

**SPDF Reader** is a secure document system designed to protect sensitive files used in **policing, cyber investigations, and confidential data handling**.

It introduces a custom file format **(.spdf)** that ensures documents can only be accessed in a **controlled and secure environment**.

## 🚀 Features

- 🔒 **AES Encryption & Decryption**  
  Protects files using strong encryption so unauthorized users cannot access data  

- 📄 **Custom File Extension (.spdf)**  
  Files cannot be opened in normal PDF readers  

- 🚫 **Restricted PDF Access**  
  Original PDFs are converted into SPDF and cannot be opened directly  

- 🎯 **Controlled File Access**  
  Limit how many times a file can be opened (view restrictions)  

- 🖥️ **Secure Viewer UI**  
  Files open only inside the SPDF Reader application  

- 🛡️ **Basic Protection Controls**  
  Reduces copying and unauthorized usage  

- 🔄 **PDF → SPDF Conversion**  
  Backend system to convert normal PDFs into secure format

## 📂 How It Works

1. Upload a normal PDF  
2. Convert it into **.spdf format (encrypted)**  
3. Open the file using **SPDF Reader only**  
4. Apply restrictions like **limited views and secure access**

## 📌 Why SPDF?

Standard PDFs are easy to copy, share, and leak.  
SPDF adds a **security layer + access control**, making it useful for **sensitive environments like policing and cybersecurity**.
